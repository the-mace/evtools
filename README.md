# evtools
This is collection of python code and tools for monitoring SolarCity, Tesla and Rivian

See help in the main tools:
* tesla.py --help
* rivian.py --help
* solarcity.py --help
    
Build and install:
python3 setup.py sdist && pip install dist/evtools-1.0.0.tar.gz
